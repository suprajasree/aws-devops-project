# AWS DevOps CI/CD Pipeline Project

## Overview

This project demonstrates an end-to-end DevOps workflow using Terraform, AWS EC2, Docker, Docker Hub, GitHub, and GitHub Actions. The application is automatically built, pushed, and deployed whenever code is pushed to the main branch.

## Technologies Used

* Python Flask
* Git & GitHub
* GitHub Actions
* Docker
* Docker Hub
* Terraform
* AWS EC2
* Linux

## Architecture

Developer → GitHub → GitHub Actions → Docker Build → Docker Hub → AWS EC2 → Docker Container → Flask Application

## Features

* Containerized Flask application
* Infrastructure provisioning using Terraform
* Docker image management using Docker Hub
* AWS EC2 deployment
* Automated CI/CD pipeline
* Automated deployment through SSH
* Linux server administration
